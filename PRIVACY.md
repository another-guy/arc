# Privacy Policy

**ARC — Workspace Bootstrap**
Last updated: 2026-05-24

## Summary

ARC is a Claude Code plugin that runs entirely on your local machine.
It does not collect, store, or transmit any personal data to external servers.

## What ARC does

ARC reads and writes files within your local workspace directory to:

- Scaffold directory and configuration structures
- Clone or pull Git repositories you explicitly configure
- Generate knowledge-base documents from your local repository contents

All operations are initiated by you and execute locally.

## Data access

ARC may read:

- Repository URLs and credentials you provide in your workspace configuration
- Source code and documentation in repositories you clone locally
- Claude Code conversation context (as any Claude Code skill does)

ARC does not read, store, or process any data beyond your workspace directory and the repositories you explicitly configure.

## Third-party services

If you configure ARC to clone repositories from GitHub or other hosting services, those requests are made directly from your machine under your own credentials.
ARC does not proxy, log, or intercept those connections.

## No analytics or telemetry

ARC collects no usage metrics, crash reports, or telemetry of any kind.
